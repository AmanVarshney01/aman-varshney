# Portfolio

Personal site built with Astro featuring projects, blog posts, and a guestbook.

## Stack

- Astro 5
- Tailwind CSS 4
- Astro DB (guestbook)
- Cloudflare adapter + Wrangler config
- Bun (package manager/runtime)

## Local Development

```bash
bun install
bun run dev
```

## Quality Checks

```bash
bun run check
bun run build
bun run format:check
```
